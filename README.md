# Competitor Analysis

A Python tool for analyzing competitor density within a geographic area defined by a polygon. It returns metrics such as number of competitors, distances, and competition level.

## Setup

```bash
make install
```

## Run

```bash
make run
```

## With custom input

```bash
make run-custom INPUT=data/sample_input.json
```

## Test

```bash
make test
```

## Type checking

```bash
make typecheck
```
